# Morse-Converter
A Morse Coverter and translator,  developed in Golang, just to test the language.
### Usage
#### Compile & Execute
if you want to compile and execute, you must be in the directory where main.go is located
```
go build
```
```
./directory name
```
For Windows OS you could execute
```
directory name.exe
```
#### Run as interpreted

```
go run main.go
```
